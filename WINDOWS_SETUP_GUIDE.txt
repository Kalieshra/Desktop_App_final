┌────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                               │
└────────────────────────────────────────────────────────────────┘

❌ "Access Denied" error
   → Right-click setup_windows.bat and select "Run as administrator"

❌ Port 8000 already in use
   → Close any program using port 8000 (check Task Manager)

❌ Chrome doesn't open automatically
   → Manually open: http://localhost:8000
   → Chrome may not be installed - will use default browser

❌ OCR not working
   → Check that tessdata folder has ara.traineddata & eng.traineddata
   → Download from: https://github.com/tesseract-ocr/tessdata

❌ Installation fails
   → Check "setup.log" file for detailed error messages
   → Try manual installation (see README_WINDOWS.md)
